# Car Rental Management App

A desktop application developed in Python for managing a car rental system.  
This project allows the management of vehicles, customers, and rental operations through a simple interface.

## Purpose

This application was developed as a learning project to practice:

- Python programming
- Data management
- Application structure
- Basic software design principles

## Features

- Vehicle management (add, edit, remove)
- Customer management
- Rental registration and tracking
- Data storage using SQLite database
- Organized resource handling

## Technologies Used

- Python
- SQLite3
- Tkinter
- CSV

## Project Structure

- `app.py` → Main application file
- `recursos/` → Images, icons or auxiliary files
- `database/` → Database files

## Notes

This project is part of a learning portfolio and demonstrates skills in Python development and basic system design.
